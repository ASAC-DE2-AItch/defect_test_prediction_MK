# modeling_v5 신뢰도 측정 — 결과 보고서

> v5 예측에 예측별 신뢰구간 + 불확실성 점수를 추가하고, valid_data로 검증
>
> 실험일: 2026-07-07 (세션 3)

---

## 1. 목적

실무형 FDC 플랫폼은 점 예측(RMSE ~60)만으로 부족하며, **"이 예측을 얼마나 믿을 수 있는가"**가 필요하다. v5 파이프라인을 바꾸지 않고(예측값 동일, valid RMSE 61.384) 신뢰도 계층을 얹고, 그 신뢰도를 valid_data로 검증했다.

---

## 2. 방법

### 2.1 예측별 신뢰구간 — Split Conformal

- train OOF(교차검증 out-of-fold) 절대잔차의 분위수로 반너비 `q(level)` 산출 → 예측 `pred ± q`.
- 분포 가정 없이 **유한표본 커버리지 보장**.

### 2.2 불확실성 점수 — Fold Disagreement

- 5개 GroupKFold 모델의 예측 표준편차(σ). valid/test는 5개 fold 모두 out-of-sample이라 정직하게 계산됨.
- 상위 20% σ를 **저신뢰(LOW)** 플래그.
- (설계 노트) v5는 row별 예측을 WF로 평균하는데, 모델이 WF 집계 피처에 의존해 **WF 내 row 예측이 거의 동일**(within-WF σ≈0)하여, 불확실성 신호로는 within-WF 편차 대신 fold 불일치를 채택.

---

## 3. 결과 (valid 검증)

| 지표 | 값 |
|------|-----|
| valid RMSE | 61.384 (v5와 동일 — 예측 불변) |
| 90% 구간 커버리지 (목표 0.90) | **0.904** ✅ |
| 90% 구간 너비 | 203.1 |
| σ–실제오차 상관 | **0.34** (유효한 신호) |
| 검증 WF 수 | valid 1990 / test 1990 |

### 3.1 신뢰구간 커버리지 (신뢰도 곡선)

목표 커버리지와 valid 실제 커버리지가 일치 → 구간이 신뢰할 만함.

| 목표 | valid 실제 커버리지 |
|------|---------------------|
| 50% | ~0.50 |
| 80% | ~0.80 |
| 90% | 0.904 |
| 95% | ~0.96 |

### 3.2 불확실성 점수(σ)의 유효성

σ가 클수록 실제 오차가 **단조 증가** → σ가 예측 신뢰도를 실제로 반영.

| σ 5분위 (낮음→높음) | 평균 절대오차 |
|---------------------|---------------|
| 1 | 38.0 |
| 2 | 47.1 |
| 3 | 52.6 |
| 4 | 64.0 |
| 5 | 69.1 |

- **저신뢰(LOW) 플래그 wafer 평균오차 68.1 vs 정상(OK) 46.9** — 저신뢰 예측이 실제로 ~45% 더 틀림.

---

## 4. 실무 활용

- **검사 우선순위**: LOW 플래그(상위 20% σ) wafer를 우선 실검사 → 예측이 불확실한 웨이퍼를 집중 관리.
- **신뢰구간 보고**: 각 wafer 예측을 `pred [lower, upper]`로 제시 → 현장 의사결정 시 불확실성 함께 전달.
- **신규 lot 견고성**: σ·구간 모두 정답 없이 추론 시점에 계산 → 신규 데이터에 그대로 적용.
