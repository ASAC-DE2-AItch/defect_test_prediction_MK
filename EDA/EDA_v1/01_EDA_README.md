# 01_EDA.ipynb 안내서

> 이 문서는 **노트북 사용 설명서**입니다. 분석 "결과"가 궁금하면 같은 폴더의 `01_EDA_REPORT.md`를 보세요.

## 이 노트북은 무엇을 하는 건가요?

반도체 제조 공정에서 수집된 센서 데이터를 **처음으로 살펴보는** 탐색적 데이터 분석(EDA) 노트북입니다.

요리를 시작하기 전에 재료 상태를 확인하는 과정과 같습니다 — 재료(데이터)가 몇 개인지, 상한 재료(결측·이상치)는 없는지, 어떤 재료가 맛(결함 수치)에 영향을 주는지 확인하고, **예측 모델의 전략 방향**을 잡습니다.

## 30초 요약 (핵심 결과 미리보기)

| 확인 항목 | 결론 |
|-----------|------|
| 데이터 품질 | 부분 결측 0개, 이상치 0개 → 매우 깨끗함 |
| 예측 단위 | C65가 웨이퍼 내 상수 → **웨이퍼(C64) 단위로 집계 후 예측** |
| 가장 중요한 센서 | **C17** (상관 -0.797), 그다음 C10/C39(0.48), C12(0.34) |
| Train↔Test 분포 | 동일 (Adversarial AUC ≈ 0.50) → 표준 파이프라인 OK |
| 버릴 컬럼 | 전부결측 8 + 상수 20 + Lot/중복 식별자 |
| 모델 방향 | 트리 기반(LightGBM) + GroupKFold(C64) |

자세한 근거·그래프는 `01_EDA_REPORT.md`에 있습니다.

## 배경 지식 (용어)

- **웨이퍼(WF)**: 반도체 칩을 만드는 둥근 원판. 하나에서 여러 칩이 나옴.
- **C64**: 웨이퍼 고유 ID (예: `C64_1`).
- **C65**: 예측 목표(결함 테스트 수치). 높을수록 결함 많음.
- **C1~C63**: 장비 센서 측정값(온도·압력·시간 등).
- **FDC**: Fault Detection and Classification. 공정 센서 데이터 수집 시스템.

## 노트북 구성 (셀별 설명)

### 셀 0: 데이터 불러오기
5개 CSV를 읽고, 유닉스 타임스탬프를 사람이 읽을 날짜/시간으로 변환합니다.

| 파일 | 설명 | 용도 |
|------|------|------|
| `train_data.csv` | 센서값 + 정답 | 패턴 학습 |
| `valid_X.csv` | 센서값만 | 중간 점검 |
| `test_X.csv` | 센서값만 | 최종 평가 |
| `valid_Y_answer.csv` | 검증 정답 | 채점표 |
| `test_Y_answer.csv` | 테스트 정답 | 채점표 |

### 셀 1: 컬럼 기본 정보 파악
68개 컬럼(파생 포함) 각각의 타입·고유값 수·결측 비율·상수 여부 확인.

### 셀 2: 결측값 패턴 분석
빈칸 컬럼과 패턴을 히트맵으로 확인 → 랜덤이면 채우기, 패턴이면 "결측 여부"를 정보로 활용.

### 셀 3: 웨이퍼 그룹 구조
웨이퍼당 행 수와, C65가 웨이퍼 내 상수인지 검증.

### 셀 4: 예측 대상(C65) 분포
히스토그램+KDE / 박스플롯 / QQ플롯 / 기초 통계로 분포 모양 확인.

### 셀 5: 시간에 따른 변화(Temporal Drift)
시간이 흐르며 C65가 변하는 추세가 있는지, Train/Valid/Test 시간대가 겹치는지 확인.

### 셀 6: 분포 변화 확인(Covariate Shift)
Adversarial Validation으로 Train↔Test 센서 분포 차이 확인. AUC 0.5면 분포 동일.

### 셀 7: 센서-결함 상관관계
피어슨/스피어만 상관으로 C1~C63 중 C65와 관련 깊은 센서 탐색 + 다중공선성 히트맵.

### 셀 8: PM(C33) 영향
PM(예방 정비) 이후 경과시간이 결함에 미치는 영향 분석.

### 셀 9: 이상치 탐지
IQR 방법으로 C65와 주요 피처의 극단값 비율 계산.

### 셀 10: 범주형 변수와 결함
C6(Recipe), C7(Step), C6×C7 조합별 결함 패턴 분석.

## 주요 분석 기법 용어

| 용어 | 쉬운 설명 |
|------|----------|
| EDA | 데이터를 처음 훑어보는 과정 |
| 히스토그램 / KDE | 값 빈도를 막대/부드러운 곡선으로 표현 |
| 박스플롯 | 분포를 상자+수염으로 요약 |
| QQ-플롯 | 정규분포와 얼마나 닮았는지 확인 |
| 상관계수 | 두 변수의 관련 강도(-1~+1) |
| 다중공선성 | 여러 변수가 비슷한 정보를 담는 문제 |
| Adversarial Validation | Train/Test 분포 차이 확인 기법 |
| IQR | 데이터 중간 50% 범위, 이상치 판단 기준 |
| 왜도/첨도 | 분포의 치우침/꼬리 두께 |
