# modeling_v3.ipynb — 노트북 안내서

> Optuna 하이퍼파라미터 자동 탐색을 적용한 v3 모델링 파이프라인
>
> 이전 버전: `modeling_pipeline.ipynb` (v1), `modeling_v2.ipynb` (v2)

---

## 실행 전 준비

1. **커널 설정**: `venv` 선택
2. **데이터 위치**: `문제1(하)/`, `문제1_하_answer/` 폴더 필요
3. **실행 시간**: 약 60~70분 (Optuna 100 trials가 대부분 차지)

---

## v1 대비 변경사항

| 항목 | v1 | v3 |
|------|----|----|
| 피처 | 315개 (글로벌 집계) | 299개 (동일 방식, 컬럼 수 차이는 더미 변수) |
| 하이퍼파라미터 | 수동 설정 | Optuna TPE 100 trials 자동 탐색 |
| learning_rate | 0.05 | 0.00576 (더 낮음 → 더 많은 트리) |
| num_leaves | 63 | 189 (더 복잡한 트리) |
| min_child_samples | 20 | 14 |
| best_iteration | ~100 | ~1,000 (10배 더 많은 트리) |
| n_estimators 상한 | 2,000 | 3,000 |
