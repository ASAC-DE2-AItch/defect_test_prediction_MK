## 8. 작업 방식 (v14 고정)
- **미러 검증 전면 금지** — 노트북·코드는 제작 즉시 전달, 실행·판정은 사용자 로컬 venv(공식).
- 방법 변경 = 새 파일 · README = 살아있는 문서(갱신) · REPORT = 스냅샷(수정 금지) · PLAN 개정 = v1.x.
- floor(R10) 5센서 = C17(온도)·C11(플라즈마)·C31(RF)·C15/C16(가스), 각 ≥1 assert 승계.

## 9. 종결 & 핸드오프
- **모델링 종료.** 추가 피처군·튜닝·재시도 없음(중단기준 도달 후 연장 금지, R7④).
- **운영 권장**: **core10 + 정기 재학습**. 재학습은 선택이 아니라 필수 — 무재학습 254.9 vs 재학습 98.4(주간 fold 기준 147→319 단조 악화).
- **팀 목표(시간축 RMSE 70)**: **도달 불가** 판정. 현 최선 98.36 과 약 28pt 격차이며, 마지막 지렛대(P1~P4)가 격차를 오히려 벌렸다. lot-CV 의 66~71 은 미래를 엿본 수치라 현업 근거로 인용 불가.
- **잔여(선택)**: same-era 1회 채점은 챔피언 부재로 판정 가치 없음 → 미실시 권장.
- **범위 밖(다음 기회)**: 웨이퍼-집계 FDC 표현 자체의 한계이므로, 원시 트레이스 수준의 다른 표현(물리 기반 상태추정 등)은 본 결론이 부정하지 않는다.
