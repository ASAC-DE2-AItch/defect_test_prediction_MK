## 0. 요건 v2.1 — 조작적 정의 (모든 판정의 기준점)

### 0.1 요구사항 → 게이트

| 요구사항 | 조작적 정의 |
|---|---|
| **현업에서 사용 가능할 것 (최우선)** | ① 정량: 신규 Lot 정직-CV 성능(아래 RMSE·R²) ② 정성: **현업 검증 배터리 B1~B5**(§7) 통과 |
| RMSE: **"현재 결합 모델보다 좋게"** | 최종 채택 모델의 **GKF(C20) OOF RMSE < 기준선 B − 0.5pt** (로컬). 기준선 B 정의는 §0.3. **v2.3 확정: 목표 = 70.712** (B=min(B0,B1)=71.212) |
| R² ≥ 0.9 | **정직축 R²** = 1 − (GKF RMSE / σ_C65)², σ=261.7. 현재 ≈ **0.928로 충족** → "0.9 밑 후퇴 금지" 가드레일 (RMSE ≤ 82.7 동치) |
| 필수 센서 C17·C11·C31·C15·C16 각 ≥ 1 | floor 제약 — fixed(core10+champion) 구조로 **자동 보장**. 모든 피처 조작 후 assert 재확인 |

**48.76의 지위**: v12 valid(인터리브)축 수치 → **참고치로 강등**. 원하면 M8 부록 R에서 1회만 채점(의사결정 사용 금지).

### 0.2 평가 축 (의사결정 언어 — v2.0과 동일)

| 축 | 정의 | 역할 |
|---|---|---|
| **① 현업축 (主)** | **GroupKFold(C20) 5-fold OOF** — 학습에 없던 Lot 예측 | 모든 선택·튜닝·게이트의 기준 |
| **①′ 보조축** | (a) **다중시드 Lot-CV**: 유니크 C20을 `KFold(5,shuffle,seed∈{1,2,3})`로 묶은 lot-fold (b) **시간분할**: C40 기준 전기 학습→후기 평가(Lot 단위 컷) | 배터리 B1·B2 — 이중딥핑 통제·미래 외삽 확인 |
| ② 참고축 | KFold(`fold_kf5`) OOF · (옵션) valid/test | **기록만, 의사결정 금지** |

**홀드아웃 규율**: 시간분할 후기 구간 **통산 2회**(M6 1회 + M8 1회). GKF는 횟수 무제한이되 채택 판정은 반드시 다중시드 lot-CV(B1)로 교차 확인.

**CV 구현 (v2.2 동결)**: 정직축 GKF 는 동결 `stable_group_kfold`(argsort `kind='stable'` 배정)를 쓴다 — 표준 sklearn GroupKFold 의 numpy 버전 타이브레이크 의존을 제거해 로컬·클라우드 어디서든 **동일 파티션**. 로컬 동결값 정확 재현(수치 불변). KFold(`fold_kf5`)는 고정 컬럼(원래 환경독립). 결정 이력 6 참조.

```
[피처셋 확정 — 모델보다 먼저]
 1) 모델별로 두 셋의 GKF 비교 (|ΔGKF| ≥ 0.3만 유의미한 차이로 인정)
 2) 두 모델이 같은 셋을 가리키면 → 그 셋 확정
 3) 갈리거나 양쪽 다 tie → 두 모델 GKF 평균이 낮은 셋. 평균 차 < 0.3 → 피처 적은 쪽
    (보조 근거: 안정 코어 분석 §1.3 — Cons-GA가 RFECV 교차생존 98%로 우세)
[모델 트랙]
 M4에서 모델은 확정하지 않는다 — 두 모델 모두 확정 셋을 들고 M5 듀얼 튜닝 진출.
 단, ET가 양쪽 셋 모두에서 GKF가 B0(71.2)보다 크게 나쁘면(> +2pt) ET 트랙 조기 종료 가능(비용 절약).
[기준선]
 B1_LGBM = 확정 셋 × LGBM 로컬 GKF (공식 기준선) · B1_ET = 동일 셋 × ET (도전자 기록)
```

## 6. Phase 2 = M5 — 듀얼 재튜닝 → 모델 챔피언 확정 (Colab 반나절 + 로컬 ~1.5h)

### 6.1 두 트랙 (확정 셋 고정, 목적함수 = GKF 5-fold OOF RMSE)

**LGBM 트랙 (Stage A: Colab 2–4h)**
- Optuna 80 trials, `num_boost_round=3000`+`early_stopping(100)`, MedianPruner(2fold 시점).
- 공간: `learning_rate` log[0.005,0.1] · `num_leaves` [31,255] · `min_child_samples` [5,100] · `feature_fraction` [0.4,1.0] · `bagging_fraction` [0.5,1.0] · `bagging_freq` [1,7] · `lambda_l1/l2` log[1e-3,10] · `min_split_gain` [0,1] · seed 42.

**ET 트랙 (Stage A: Colab 1–2h)**
- Optuna 40~60 trials (fit이 싸므로 pruner 없이도 가능).
- 공간: `n_estimators` [300,800] · `max_depth` {8,16,24,32,None} · `min_samples_leaf` [1,20] · `min_samples_split` [2,20] · `max_features` [0.3,1.0] · `random_state` 42. 대치 = median 고정(변형은 사다리에서).
- KFold는 두 트랙 모두 **기록만**.

**Stage B — 다중시드 스크리닝 (Colab ~2h)**: 트랙별 상위 3 trial × lot-CV seed{1,2,3} 재평가 → 트랙별 대표 1개 선발(3시드 평균 최소 ∧ 최악 시드 B1_LGBM 대비 악화 없음).

**Stage C — 로컬 확정 (~1.5h)**: 트랙 대표 2개(LGBM 1·ET 1)를 로컬에서 GKF + lot-CV 1시드 재측정 → 수치 동결.

### 6.2 모델 챔피언 규칙 (사전 등록)

```
 1) 로컬 GKF 낮은 모델 = 챔피언  (|Δ| ≥ 0.3)
 2) |Δ| < 0.3 → 다중시드 lot-CV 분산(std) 작은 쪽
 3) 그래도 tie → LGBM (운영 연속성·gain 해석 용이·결측 자체 처리)
 준우승 모델·파라미터는 폐기하지 않고 동결 보존 → F5′(앙상블) 재료
```
