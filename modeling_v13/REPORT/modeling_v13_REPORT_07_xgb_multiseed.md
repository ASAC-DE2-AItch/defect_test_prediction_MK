확정 셋 **Conservative-GA(99)** 에서 **XGBoost 가 LGBM 을 4개 파티션(stable + seed 1/2/3) 전부 이겼다** — 시드 Δ(LGBM−XGB) **평균 +0.590 · 최악 +0.426**. 사전등록 **§6.2 기준(평균 ≥0.5 ∧ 최악 ≥0) 양쪽 통과 → XGBoost M5 챔피언 후보 편입 근거 성립(R7 확정)**. 단 이는 **XGB vs LGBM 상대 우위**의 확정이며, **게이트 70.712(v2.3) 통과는 M5 튜닝 챔피언 대상**(별개).

## 2. 정합 확인 ✅

- **LGBM 4파티션 전부 미러와 Δ0.000 일치** — 다중시드 파티션은 numpy legacy RandomState(버전 안정), stable 은 v2.2 폴드맵 → 환경독립. LGBM 결정적.
- **XGBoost 로컬 > 미러 +0.01~0.26** (로컬 xgboost 버전 ≠ 컨테이너 3.2.0 — R6). 그래도 **전 파티션 XGB < LGBM** 불변.
- **절대 GKF 파티션 편차 큼**(LGBM 69.87~73.23) — 파티션 난이도 차. **그러나 상대 Δ 일관 양(+0.43~0.92)** → 다중시드의 필요성·stable 앵커 고정(v2.3)의 정당성 재확인.

## 4. 정직한 유보

1. **상대 우위는 확정, 그러나 양쪽 미튜닝** — LGBM=M8_PARAMS(공격적, num_leaves 175) · XGB=M8 근사. 이 우위엔 **LGBM 과적합 핸디캡**이 섞였을 수 있음. **M5 에서 둘 다 GKF 목적 튜닝**해 공정 재대결해야 최종 챔피언 확정.
2. **게이트 70.712 미달(원값)** — XGB stable 70.773 · 시드평균 70.745 = **앵커 바로 위**(0.03~0.06 미달). 원값으론 G5 불통 → **M5 튜닝으로 70.712 아래로 내려 증명**.
3. **ET 는 정직축 최약체(R1)** — M4 GKF 72.4(LGBM +1.0 열세). M5 챔피언 후보에서 **제외하고 F5′ 앙상블 재료로만 보존** 권고(비용 집중).
