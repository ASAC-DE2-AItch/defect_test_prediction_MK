## 4. 터미널 함의 (정직)

1. **재학습은 필수이자 효과적 (i).** 무재학습은 배포 불가(pooled 254.9·발산). 재학습이 2.5배 회복(→99.8). 비정상 공정의 정공법이 재학습임을 실증.
2. **⚠️ FDC 센서는 미래-Lot 예측에 운영 가치 없음 (ii) — v13 핵심 테제 반증.** 재학습해도 **85개 센서 피처가 백본 10개(core10)를 못 이김**(오히려 −1.48). v13의 센서 복원(C11·C15·C16·C31)·655 풀·GA 선별·챔피언 XGB(−4.6)가 **운영축(미래-Lot)에서 미실현.** 앞선 신호의 최종 확증: M4.5(잔차 센서 무상관 0.037) → core10 ablation(센서 단독 약함) → REPORT_11(센서–타깃 관계 시간 비정상) → REPORT_12(드리프트 프록시 lot-CV 무가치) → **본 리포트(운영 무가치).**
3. **정확도 바 미달 (iii).** 재학습해도 R² 0.85(미래 1주 예측). 오차가 레짐-온셋 주에 집중 = *새 레짐은 보기 전엔 못 맞춘다*(M4.5 Lot 천장의 시간판). pooled 사전등록 판정 = 0.9 미달.

## 6. 핸드오프 (정직 · 달성 가능 최선)

- **운영 권장 모델 = core10(10피처) + 주간(또는 PM 경계) 재학습.** 센서 불요(운영축 무가치). R²≈0.85·pooled RMSE≈98, 무재학습 대비 2.5배 우위.
- **한계 명시**: 레짐-온셋 주(PM 직후)는 어떤 모델도 예측 불가 — 재학습 후 회복. 이 구간은 "예측 신뢰 낮음" 플래그 운영.
- **모니터링**: 입력 PSI · 잔차 lot-ICC · 주간 RMSE · PM 이벤트(pm_log) 트리거 재학습.
- **v13 자산 용도 재정의**: 챔피언 XGB(99)·lean-85는 **"같은 시기 새 Lot"**(후향 분석·대회축)용으로만 유효. **미래 배포엔 core10.**
- **잔여 옵션(비의사결정)**: 부록 R 대회축 참고 채점(valid/test 1회)은 기록용으로 가능(§0 의사결정 불사용).
