## 4. 정직한 발견 (진단)

1. **M5 −4.6은 "같은 시기 새 Lot" 한정 이득이다.** B1이 재확증(챔피언 66.98 vs 기준선 71.34). 그러나 lot-CV는 랜덤 lot 분할 = 학습·평가가 **같은 기간을 공유**한다.
2. **미래 Lot엔 전이 실패 = 시간 드리프트.** 시간분할(2019-01 예측)에서 챔피언 66.7 → 77~105. 학습기(2018-12)와 예측기(2019-01)의 분포가 달라, 99피처 튜닝 모델이 외삽하지 못한다. 70/30 컷에선 **안정 백본 core10이 오히려 이긴다** — 센서 다수 피처가 미래엔 불안정.
3. **B2가 설계 목적 그대로 작동했다.** 강건이 M5 판정에서 "−4.6이 미래에도 버티는지 = B2 최종 심판"이라 **사전 기록** → B2가 실패를 잡았다(체리피킹·자기기만 방지). 정직축의 정당성 자기검증.
4. **M4.5 천장과 정합.** 남은 Lot 효과가 센서로 접근 난망(0.037) + 이제 시간축 드리프트까지 → 센서 bulk 피처(B4 필수센서 12.7%뿐, 나머지는 C25 노후드리프트 등)가 미래엔 오히려 독이 됨.
5. **⚠️ 축 불일치 경고 (M7 관련).** M7 사다리의 채택 기준은 **lot-CV**(§8.1)인데, 실패한 축은 **시간분할(B2)**. lot-CV를 더 개선해도 B2가 따라온다는 보장이 없다. 게다가 홀드아웃 규율(R5)상 시간분할 조회는 **M8 1회만** 남는다 → M7을 lot-CV로 돌린 뒤 M8에서 단 1회 시간분할 재판정 = **고위험**.
