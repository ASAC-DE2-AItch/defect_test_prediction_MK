## 3. 판독 (G4.5 — 판정 아님, 캘리브레이션)

- **사전등록 ICC 규칙**: 잔차 ICC 0.784 ≥ 0.4 → "여지 큼 (F1′/F4′ 적극)" — *노트북 산출값 그대로 기록*.
- **진단3 정밀화(사전등록)**: 그 여지가 센서와 거의 무상관(≤0.037) → F1′(기존 센서 교호)·F4′(같은 센서 신규 집계)의 **실제 기대이득은 낮다**. 남은 Lot 효과는 현 FDC 트레이스 **밖** 정보(다른 센서·계측·소재 이력 등)에 있을 가능성.
- **천장 추정**: 통계적 개선 여지 大 · 현 센서 접근성 小 → **정직축 −0.5pt는 불확실~난망**.
- **정직한 반례 여지**: 상관은 선형·주변 지표. 트리는 비선형·교호를 쓰므로 0.037이 "신호 0"의 증명은 아니다. 최종 판단은 M5·(필요 시) F1′ 실측으로.

## 4. 함의 (M5 / M7 캘리브레이션)

- **M5(파라미터 재튜닝) = 1순위**: 10피처용 M8_PARAMS를 99피처에 맞게 풀면 정직축·KFold 모두 내려갈 여지 — **센서 접근성과 무관한 지렛대**.
